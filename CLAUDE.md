# Claude AI 작업 가이드

이 문서는 Claude AI가 M 프로젝트에서 작업할 때 참고해야 할 가이드입니다.

## 프로젝트 개요

M 프로젝트는 Unity 기반의 프로젝트로, Akasha 반응형 상태 관리 시스템을 활용합니다.

## 작업 시작 전 필수 참조 문서

### 1. 기획 및 설계
- **M_Project_Design_Plan.md**: 프로젝트의 전체 기획 및 설계 문서입니다.
  - 작업을 시작하기 전에 반드시 이 문서를 읽고 프로젝트의 방향성과 요구사항을 파악하세요.
  - 새로운 기능을 추가하거나 수정할 때는 이 기획 문서와 부합하는지 확인하세요.

### 2. 코딩 작업
코딩 작업을 수행하기 전에 다음 두 문서를 반드시 읽으세요:

- **Akasha_API.md**: Akasha 시스템의 API 문서
  - Akasha의 모든 클래스와 메서드에 대한 상세한 설명이 포함되어 있습니다.
  - RxState, RxComputed, RxAction 등 핵심 API의 사용법을 확인할 수 있습니다.

- **Akasha_README.md**: Akasha 시스템의 개요 및 사용 가이드
  - Akasha의 기본 개념과 철학을 이해할 수 있습니다.
  - 실제 사용 예제와 패턴이 포함되어 있습니다.

## 작업 순서

1. **M_Project_Design_Plan.md** 읽기 - 요구사항 및 기획 파악
2. **Akasha_README.md** 읽기 - 시스템 개념 이해
3. **Akasha_API.md** 읽기 - API 사용법 숙지
4. 코드 작성 시작

## 주의사항

- Akasha 시스템의 반응형 패러다임을 준수하세요.
- 상태 관리는 RxState를 통해, 파생 값은 RxComputed를 통해 관리하세요.
- 부작용이 있는 작업은 RxAction을 사용하세요.
- 코드 작성 시 기획 문서의 요구사항을 항상 염두에 두세요.

## 문서 업데이트

이 프로젝트에서 작업하면서 중요한 패턴이나 가이드라인을 발견하면 이 문서에 추가해주세요.
